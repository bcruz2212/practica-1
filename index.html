Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea. git status: Este comando se usa para obtener el estado actual del repositorio git clone: reclamar repositorio de URL en ubicacion distinta. git pull:hacer el merge de los commits de otro branch. git checkout: cambiar a otro branch git log:mostrar todos los commits del branch history. git branch:lista de los branches git add: agregar el file antes del siguiente commit. git commit:agregar nuevas lineas con su comentario. git push: transferir commits locales a repositorio remoto. git merge: Unir un branch con el actual.
